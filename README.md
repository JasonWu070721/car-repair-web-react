# Car Repair WEB

## Features

- OS: Linux, Windows
- Framework: React.js, Material UI, OpenAPI
- protocol: RESTful API
- Language: Typescript
- Container: Docker
- Cloud: AWS

## Quick start

### Docker

#### Run on Dockerfile

```shell script
docker build -t car_repair_web .
```

#### Run on docker-compose

```shell script
docker-compose down

docker-compose up
```
